# CG-portfolio
CG modeling portfolio created with Maya

## ワインボトルと机
### ファイル名
Practice.png
### 使用ソフト
Maya
### 制作日数
1日
### 工夫した点
曲線の表現を工夫した。ワイングラスやワインボトルは、EPカーブツールを用いて表現した。机の角の丸みは円に沿うようにトレースして実現した。
### 改善点
EPカーブツールなどのカーブツールを使うと色を塗る時少しやりづらいと感じるため使い方を改善する。また、ライトの表現ができていないので次の作品で追加し理解する。

## ボーリングブース

### ファイル名
Bowling.png
### 使用ソフト
Maya
### 制作日数
2日
### 工夫した点
ライトの当たり方による違和感のない影を作った点。レイトレーシングを理解して、光の反射率などの詳細を調節してより自然に見えるように制作した。
